## Quant Developer Assignment


### Overview
Real-time analytics system for Binance Futures ticks with live statistical arbitrage analytics.


### Stack
- Python
- Streamlit
- WebSockets
- Pandas
- Statsmodels


### Analytics
- Hedge ratio (OLS)
- Spread & Z-score
- Rolling updates
- ADF stationarity test
- Alerts


### AI Usage
ChatGPT was used for structuring, boilerplate, and validation of analytics formulas.
